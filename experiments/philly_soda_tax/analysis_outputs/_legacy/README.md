# Legacy / superseded analysis outputs

Plots and CSVs parked here on 2026-05-11 when the
Philly analysis pivoted to symmetric 18-month windows
with passage (Jun 2016) as the cut date. Everything in
this folder is regenerable from the analyzer scripts in
`../../code/`.

Why each is here:

- **`did_18mo_B_effective_jan2017_*`** — Spec B from the
  18-month DiD (Jan 2017 effective-date cut). Set aside
  because the user picked passage (Spec A) as THE
  treatment date for now. Implementation is shown as a
  vertical reference inside the post period instead of a
  separate spec.
- **`event_study_B_effective_jan2017_*`** — same reason.
- **`did_summary.csv`, `period_means.csv`** — outputs of
  `analyze_national_panel.py`'s long-window DiD (2010 +
  2014-pre vs 2017-2023-post). Superseded by the
  symmetric 18-month DiD, which is the headline now.
- **`monthly_spend.png`, `monthly_spend_by_arm.csv`,
  `pre_post_summary.csv`** — outputs of
  `analyze_philly_diff.py`, the original single-DMA
  analyzer. Useful for the early exploratory phase but
  not the headline spec.
- **`monthly_spend_top4_byparent.png`,
  `monthly_spend_top4_total.png`** — outputs of
  `analyze_philly_top4.py`. Single-DMA top-4 view from
  2014-2018 data; superseded once the national panel
  landed.
- **`monthly_spend_coke_seasonal.png`** — earlier
  Coke-only seasonal plot, replaced by the Coke+Pepsi
  two-panel `monthly_spend_coke_pepsi_seasonal.png` in
  the main folder.

All files here can be regenerated by re-running the
relevant analyzer with the older config. Kept for
reference, not for further analysis.
